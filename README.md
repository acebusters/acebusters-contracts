### installation

```
npm install
```

### run tests

```
npm test
```

or run individual tests like this:

```
npm test test/table.js
```

### code coverage

```
npm run coverage
```

Executes instrumented tests on a separate testrpc. Coverage report is dumped to the terminal and into `coverage/index.html` (Istanbul HTML format)
